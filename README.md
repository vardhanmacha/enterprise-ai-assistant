# Enterprise AI Assistant

An AI-powered enterprise assistant integrating:
- SQL data querying
- Machine learning predictions
- FastAPI backend APIs

## Features
- Natural language enterprise data insights
- Expense prediction model
- Chatbot-style assistant
- Decision support automation

## Tech Stack
- Python
- FastAPI
- SQLite
- Pandas
- Scikit-learn

## Example Queries
- Show department expenses
- Predict next month costs
- Which department has highest spending?

## Status
Project in development.
## Architecture

User Query
↓
FastAPI Backend
↓
SQL Data + ML Prediction
↓
Insight Response
